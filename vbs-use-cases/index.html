VBS · Mapa de use cases por departamento
Cómo usar este mapa: Cada caja es un departamento de VBS con sus use cases candidatos. Click sobre cada item para priorizarlo dentro de su departamento: primer click = #1 del departamento, segundo click = #2, etc. Cada departamento tiene su propia secuencia, lo que abre el juego para mapear cargas de trabajo por área. El sidebar agrupa la lista priorizada por departamento. Drag-and-drop reordena dentro del mismo departamento. El botón ✎ permite sumar comentarios. La caja "Otros items" suma use cases que no estén listados.
